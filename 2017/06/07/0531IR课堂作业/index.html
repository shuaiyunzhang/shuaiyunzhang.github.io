0531IR课堂作业

发表于 2017-06-07   |  

开始时，借用同学的阿里云账号登陆，找了半天在Mac使用的SSH客户端，最后回归到Mac本身的Terminal。但新问题又来了，在输入用户名和密码后，一直显示Permission denied，在这个地方找了很久的原因。

最后仍没有解决问题，猜测是公钥权限的问题，但不是很清楚，比较复杂放弃。所以自己购买了阿里云，然后问题又来了，在登陆的时候，输入了用户名（root）和密码，还是显示login incorrect，原因未知，至少以前使用云时没遇到过这种问题。为了节省时间，选择还是在本地操作，当然云上的问题解决同步进行，这里主要记录本地的操作。

Java安装

这里没遇到什么问题，云上也是自带的，均为最新的java版本，带过。

Solr的安装

这里下载的是solr-6.5.1的tgz版本，下载地址为官网提供的镜像地址，下载之后解压，参照老师上课给的资料以及官网文档，进入solr管理界面。

$ cd solr-6.5.1/
$ bin/solr start -e techproducts

构建查询

进入对应core的查询界面

可以直接在图形化界面操作，也可以构建查询表达式，两者的效果是等效的。在地址栏直接输入

http://localhost:8983/solr/techproducts/select?indent=on&q=*:*&wt=json

分面功能的使用

要使用分面首先得在Query界面启用分面，即facet的值为true，然后就是其他facet参数的设置。比如查找价格范围为0到400的产品，并按类别对结果进行分组。

http://localhost:8983/solr/techproducts/select?q=price:[0 TO 400]&fl=id,name,price&facet=true&facet.field=cat

在命令行输入

curl 'http://localhost:8983/solr/techproducts/select?wt=json&indent=true&q=*:*&rows=0'\
'&facet=true&facet.field=manu_id_s'
